The problem

Many educational institutions still depend on manual attendance systems, which are time-consuming and error-prone. Teachers spend a significant portion of class time marking attendance, reducing valuable instructional hours. Students often waste free periods with unproductive activities due to lack of structured guidance. This leads to poor time management and reduced alignment with long-term goals. There is also a gap in personalized learning support during idle classroom hours. Institutions currently lack tools that integrate daily schedules with individual student planning and automated tracking.

The solution (high level)

  1. Automated Attendance — QR check-in, Bluetooth or NFC-based quick scans for fast, accurate records.
  2. Idle-Hour Planner — Structured micro-tasks and personalized suggestions during free periods.
  3. Schedule Integration — Sync daily timetables with individualized study and career micro-goals.
  4. Reports & Analytics — Attendance stats, participation heatmaps and time-management suggestions for students and teachers.

How to deploy to GitHub Pages

  1. Create a new GitHub repository (public or private)
  2. Upload these files (or push via git)
  3. Go to repository Settings → Pages, and set the branch to main (or gh-pages) and root folder /
  4. Save — your site will be live at https://<username>.github.io/<repo>

See the included README.md for direct git commands.
